# brcovid19api

Simple API that providing current covid19 status in Brazil. Used as toy problem in order to hands on k8s and Go.

## tl;dr

```bash
curl http://api.montenegrodr.com/covid19 

{
  "confirmed": 1960,
  "deceased": 34,
  "recovered": 2,
  "updatedAt": "0001-01-01T00:00:00.000Z"
}

```

## Build

Requirements:

- Go
- go-swagger
- Python 3
